# TapTap 🟢

A Android reaction game. Tap the correct (green) button as fast and as often as you can — before the 30 seconds run out!

## 📱 Gameplay

- A 2×2 grid of buttons appears on screen
- One button turns **green** — tap it!
- Each correct tap = +1 point
- Tap the wrong button → game over
- Time runs out after **30 seconds** → game over
- Your **high score** and **last round score** are saved automatically

## 🛠️ Tech Stack

- **Language:** Java
- **Platform:** Android (minSdk 24 / Android 7.0+)
- **Build system:** Gradle
- **UI:** XML Layouts (no third-party UI libraries)

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest recommended)
- Android SDK 29+
- Java 8+

### Build & Run
1. Clone the repository:
```bash
   git clone https://github.com/felix-richter/TapTap.git
```
2. Open the project in **Android Studio**
3. Let Gradle sync
4. Run on an emulator or physical device (Android 7.0+)

## 📦 Download

A pre-built APK is available on the [Releases page](https://github.com/felix-richter/TapTap/releases).

Simply download the `.apk` file and install it on your Android device (Android 7.0+).
